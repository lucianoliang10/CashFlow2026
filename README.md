# CashFlow 2026

Projeto Streamlit para projeção de caixa mensal de Janeiro/2026 a Dezembro/2026, com edição inline dos itens de outflow e recálculo automático.

## Como rodar localmente

```bash
streamlit run app.py
```

## Deploy no Streamlit Cloud

1. Suba este repositório para o GitHub.
2. No Streamlit Cloud, selecione o repositório e a branch `main`.
3. Defina **Main file path** como `app.py`.
4. Clique em **Deploy**.
